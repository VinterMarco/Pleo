# PLEO - Finance Tracking App

iOS App built using Swift, SwiftUI, Firebase.

Builded for iOS 17.

## Main Features

- Expense Tracking
- Budget Management
- Transaction History
- Financial Goals
- Reports and Analytics
- Multi-Currency Support
